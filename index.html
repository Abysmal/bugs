Simple Issue Tracking for Teams

Download this project as a .zip file Download this project as a tar.gz file

Features

  • As few features as possible.
  • Issues can be redacted using Markdown
  • Issues can be grouped using a flexible tag system to match the way you think
  • Attachement upload
  • Default Assignee per project
  • Several roles & privileges:
    • User: can only read the issues in the projects they are assigned to
    • Developer: can update issues in the projects they are assigned to
    • Manager: can update issues in all projects, even if they aren't assigned
    • Administrator: can update all issues in all projects, create users and view administration

Installing Little BIT v1.0

  • Create a MySQL Database
  • Make /uploads/ write-able (CHMOD 777)
  • Open /install/ in your browser
  • Delete or rename /install/

Requirements:

  • Tested on: Apache, IIS
  • PHP 5.3+
  • MySQL 5+
  • PDO Extension for PHP (MySQL)
  • MCrypt Extension for PHP
  • Javascript Enabled - Bugs also uses heavy Javascript to make it easier to use
